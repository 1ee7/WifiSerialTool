******************************************
** 本项目属于一个基于无线的串口调试工具 **
******************************************
    /* 初始版本 V0.1 时间2020.06.12 */
******************************************
## 1. 功能
   1> 两个串口通过无线（ESPNOW透传）传输串口信息，可读写。
   2> 可配置波特率。(带显示菜单和选择按钮--旋转编码器)
   3> 上电自动匹配另一个串口模块。
 
## 2. 硬件
   1> 主控采用esp32 4M Flash
   2> 显示采用0.96'' OLED
   3> 按钮采用旋转编码器 SIQ-02FVS3
  

## 3. 软件
   1> 基于IDF4.4 开发 
   2> 串口、ESPNOW 
   3> I2C  

## 4. 开发工具
   1> VScode
   2> Kicad
